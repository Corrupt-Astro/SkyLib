SkyLibrary Documentation

A guide to using the SkyLibrary UI framework built in Lua.

Core Concepts

SkyLibrary is designed to be a simple and efficient way to create graphical user interfaces. The main components are `Window`, `Tab`, and `Section`, which serve as containers for other UI elements.

  • Window: The top-level container for your entire UI. All other elements must be placed inside a window.
  • Tab: A container for different pages of content within a `Window`.
  • Section: A container for groups of related controls within a `Tab`.

Getting Started

To begin, you must require the SkyLibrary library and call the `Create` function to create a new window. 

local SkyLibrary = loadstring(game:HttpGet("https://raw.githubusercontent.com/yofriendfromschool1/aw-temp-fix/refs/heads/main/skui.lua",true))()

local Window = SkyLibrary.Create("My Window", true)

The `Create` function takes two arguments:

  • Title (string): The title of your window.
  • Toggleable (boolean, optional): If `true`, the window can be toggled on/off with a keybind. The default keybind is `RightControl`.

Window Methods

Once you have created a window, you can add tabs to it.

`Window:CreateTab(name)`

  • Creates a new tab within the window.
  • `name` (string): The name of the tab that will appear on the UI.
  • Returns: The newly created `Tab` object.

Tab Methods

Tabs are where you organize your content. Tabs can contain sections.

`Tab:CreateSection(name, width)`

  • Creates a new section within the tab.
  • `name` (string): The title of the section.
  • `width` (number, optional): The width of the section. Default is `400`.
  • Returns: The newly created `Section` object.

Section Methods

Sections hold the actual interactive UI elements. This is where you can add buttons, toggles, sliders, and more.

`Section:CreateButton(name, callback, tooltip)`

  • Creates a clickable button.
  • `name` (string): The text displayed on the button.
  • `callback` (function): A function to be executed when the button is clicked.
  • `tooltip` (string, optional): A description that appears on hover.

`Section:CreateToggle(name, default, callback, tooltip)`

  • Creates a toggle switch (checkbox).
  • `name` (string): The text next to the toggle.
  • `default` (boolean): The initial state of the toggle (`true` for on, `false` for off).
  • `callback` (function): A function that is called when the toggle state changes. It is passed one argument, the new state (`true` or `false`).
  • `tooltip` (string, optional): A description that appears on hover.

`Section:CreateSlider(name, min, max, default, rounding, callback, tooltip)`

  • Creates a slider to select a numerical value.
  • `name` (string): The text label for the slider.
  • `min` (number): The minimum value of the slider.
  • `max` (number): The maximum value of the slider.
  • `default` (number): The initial value of the slider.
  • `rounding` (number, optional): The number of decimal places to round to. Default is `0`.
  • `callback` (function): A function called when the slider value changes. It is passed one argument, the new value.
  • `tooltip` (string, optional): A description that appears on hover.

`Section:CreateLabel(name)`

  • Creates a non-interactive text label.
  • `name` (string): The text to display.

`Section:CreateInput(name, text_filter, default, callback, tooltip)`

  • Creates a text input box.
  • `name` (string): The label for the input box.
  • `text_filter` (string): A text filter for the input.
  • `default` (string, optional): The initial text in the input box.
  • `callback` (function): A function called when the input text changes. It is passed one argument, the new text.
  • `tooltip` (string, optional): A description that appears on hover.

`Section:CreateDropdown(name, list, default, callback, tooltip)`

  • Creates a dropdown menu.
  • `name` (string): The label for the dropdown.
  • `list` (table): A table of strings for the dropdown options.
  • `default` (string, optional): The initial value.
  • `callback` (function): A function called when the selected option changes. It is passed one argument, the new value.
  • `tooltip` (string, optional): A description that appears on hover.

Example Usage

Here is a full example of how to create a basic UI with a window, a tab, a section, and a button. 

-- Load the SkyLibrary library
local SkyLibrary = loadstring(game:HttpGet("https://raw.githubusercontent.com/yofriendfromschool1/aw-temp-fix/refs/heads/main/skui.lua",true))()

-- Create the main window
local MyWindow = SkyLibrary.Create("SkyLibrary", true)

-- Create a tab inside the window
local MainTab = MyWindow:CreateTab("Main Tab")

-- Create a section inside the tab
local MySection = MainTab:CreateSection("General Settings")

-- Add a button to the section
MySection:CreateButton(
    "Click Me!",
    function()
        print("Button was clicked!")
    end,
    "This button prints a message to the console."
)

-- Add a toggle switch
MySection:CreateToggle(
    "Enable Feature",
    false,
    function(state)
        if state then
            print("Feature enabled!")
        else
            print("Feature disabled.")
        end
    end
)

-- Add a slider
MySection:CreateSlider(
    "Set Value",
    0,
    100,
    50,
    1,
    function(value)
        print("Slider value is: " .. value)
    end
)
